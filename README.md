## my-plugins介绍

```
whatweb的插件目录
下载后放在whatweb.exe或whatweb.rb同级目录即可.
```

## Novafinger介绍

```
Whatweb的python3命令行封装脚本

添加输出格式--log-csv,用于生成json并导出为csv格式。

除使用-wh用于调用whatweb默认的-h外，复用whatweb所有原生参数
```
